# Installation

## Windows Installation

1. Download the latest Cascade release from the [Version Repository](versions.md).
2. Unzip the downloaded file.
3. Run `Cascade.exe` or the Cascade launcher.
4. If Windows warns that the app is from an unknown publisher, choose **More info** and then **Run anyway**.

## Requirements

- Windows 10 or later
- Java runtime bundled with Cascade, or Java 21 if using a development build
- Sufficient RAM for large microscopy images
